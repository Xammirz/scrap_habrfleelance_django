Парсинг данных с habr.freelance. Парсятся почти все данные, такие как: url, title, теги, основной текст, информация о юзере, хабы, время, картинка, к-во голосов, к-во избранных, контакты юзера, карма юзера и рейтинг юзера. После чего при запуске файла emails.py по почте идет рассылка людям, подписанных на рассылку новостей. 

Образец django
Скелет (структура)
Python
Установите Python версии 3.6 и выше.

Зависмости
Установите все зависмости из файла requirements.txt

Запуск программы
Для запуска программы используйте команду 'python ./manage.py runserver'
